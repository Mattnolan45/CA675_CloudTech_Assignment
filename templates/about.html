CA675 Cloud Computing 2021/22

This webapp was developed by group 21 for assignment 2. This project aims to solve a real world problem (below) by leveraging big data technologies. To achieve this a dataset of ~8GB was cleansed and loaded into a Hadoop cluster. Hive was then leveraged to analyse the data in a meaningful way. To serve this web app the flask framework was utilised. This technology uses the Pyhthon language and allows us to utilise PyHive to interact with our cluster.

While tools exist to find the best value or most environmentally friendly flight there are scenarios where the most important metric is reliability.
